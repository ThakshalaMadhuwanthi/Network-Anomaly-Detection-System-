# Network Anomaly Detection System

## Overview
This project is a Machine Learning-based Intrusion Detection System (IDS) that detects anomalous network traffic. The system uses Random Forest and XGBoost models to classify network activities as Normal or Anomaly.

## Features
- Network traffic anomaly detection
- Random Forest and XGBoost models
- Web-based user interface
- Real-time prediction
- AbuseIPDB integration for IP reputation checking

## Technologies Used
- Python
- Scikit-learn
- XGBoost
- Pandas
- Streamlit
- HTML/CSS

## Dataset
The dataset was obtained from Kaggle and contains network traffic features used for training and testing the machine learning models.

## Model Performance
- Random Forest Accuracy: 99.39%
- XGBoost Accuracy: 98.40%
